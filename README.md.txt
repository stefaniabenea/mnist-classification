# MNIST Digit Classifier (PyTorch)

This project trains a simple neural network using PyTorch to classify handwritten digits from the MNIST dataset.

## Structure
- `mnist_classification.ipynb`: Jupyter notebook with all steps: data loading, training, evaluation and visualization.

## Model
- 2-layer feedforward neural network
- Trained with cross-entropy loss and Adam optimizer

## Accuracy
- Achieved over 97% test accuracy

## Requirements
- Python 3.9+
- PyTorch
- Jupyter Notebook
